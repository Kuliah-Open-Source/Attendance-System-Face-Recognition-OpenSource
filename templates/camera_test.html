Troubleshooting Guide

Chrome/Edge:
  • Click camera icon in address bar
  • Select "Always allow"
  • Refresh the page
Firefox:
  • Click shield icon
  • Enable camera access
  • Reload page

Hardware Check:
  • Reconnect USB camera
  • Try different USB port
  • Check device manager
Software Check:
  • Close other camera apps
  • Update camera drivers
  • Restart browser
